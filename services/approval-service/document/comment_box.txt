1. View last approval with comment:

sqlite3 approval_service.db "SELECT requester_name, status, approver_comment FROM approval_requests ORDER BY created_at DESC LIMIT 1;"

2.View all approvals with comments (formatted):

sqlite3 -header -column approval_service.db "SELECT id, requester_name, status, approver_comment, created_at FROM approval_requests ORDER BY created_at DESC LIMIT 5;"

3.View only approvals that have comments:

sqlite3 -header -column approval_service.db "SELECT requester_name, status, approver_comment, processed_at FROM approval_requests WHERE approver_comment IS NOT NULL AND approver_comment != '';"

4.View all columns for the last approval:

sqlite3 -header -column approval_service.db "SELECT * FROM approval_requests ORDER BY created_at DESC LIMIT 1;"

5.Count approvals by status:

sqlite3 approval_service.db "SELECT status, COUNT(*) as count FROM approval_requests GROUP BY status;"
