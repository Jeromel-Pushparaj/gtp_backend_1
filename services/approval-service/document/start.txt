Start Kafka with Docker and Colima
colima start

Check if you have a docker-compose file
docker-compose up -d


Verify Kafka is running
docker ps


Check Kafka logs
docker logs kafka

View Kafka Topics
docker exec -it kafka kafka-topics --list --bootstrap-server localhost:9092

Monitor approval.requested topic
docker exec -it kafka kafka-console-consumer \
  --bootstrap-server localhost:9092 \
  --topic approval.requested \
  --from-beginning

Monitor approval.completed topic
docker exec -it kafka kafka-console-consumer \
  --bootstrap-server localhost:9092 \
  --topic approval.completed \
  --from-beginning

Monitor action.executed topic
docker exec -it kafka kafka-console-consumer \
  --bootstrap-server localhost:9092 \
  --topic action.executed \
  --from-beginning


Monitor action.rejected topic
docker exec -it kafka kafka-console-consumer \
  --bootstrap-server localhost:9092 \
  --topic action.rejected \
  --from-beginning

View all messages in a topic with formatting
docker exec -it kafka kafka-console-consumer \
  --bootstrap-server localhost:9092 \
  --topic approval.requested \
  --from-beginning \
  --property print.key=true \
  --property print.timestamp=true



Check Kafka Consumer Groups
docker exec -it kafka kafka-consumer-groups \
  --bootstrap-server localhost:9092 \
  --list
