# COMPLETE APPROVAL WORKFLOW TEST

## IMPLEMENTATION COMPLETE!

You now have a complete HTTP endpoint that:
1. Accepts channel_name and approver_name (not IDs)
2. Looks up the channel and users
3. Publishes to Kafka topic `approval.requested`
4. Approval Service consumes from Kafka
5. Sends Slack message with Approve/Reject buttons
6. Human clicks button in Slack
7. Socket Mode receives the click
8. Updates database
9. Publishes to Kafka topic `approval.completed`

## STEP 1: Restart the Service

Stop the current service (Ctrl+C) and restart it:

```bash
cd gtp_backend_1/services/approval-service
go run cmd/main.go
```

You should see the new endpoint in the logs:
```
POST /api/v1/approval/request             - Create approval request (publishes to Kafka)
```

## STEP 2: Test the Complete Workflow

### Send Approval Request via HTTP

```bash
curl -X POST http://localhost:8083/api/v1/approval/request \
  -H "Content-Type: application/json" \
  -d '{
    "channel_name": "channel_approval_service",
    "approver_name": "Sarumathi S",
    "requester_name": "John Doe",
    "request_type": "budget_approval",
    "message": "Please approve budget request for $5000",
    "request_data": {
      "amount": 5000,
      "department": "Engineering"
    }
  }'
```

**Expected Response:**
```json
{
  "success": true,
  "message": "Approval request created and published to Kafka",
  "request_id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
}
```

## STEP 6: Check Kafka (approval.completed topic)

Open a new terminal and consume from the completion topic:

```bash
docker exec -it kafka kafka-console-consumer \
  --topic approval.completed \
  --bootstrap-server localhost:9092 \
  --from-beginning
```

You should see a message like:
```json
{
  "request_id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "status": "approved",
  "approved": true,
  "processed_by": "Sarumathi S",
  "processed_at": "2026-03-01T13:30:00Z",
  "reason": "",
  "request_data": {
    "amount": 5000,
    "department": "Engineering"
  }
}
```

## Additional Test Cases

```bash

  -H "Content-Type: application/json" \
  -d '{
    "channel_name": "non-existent-channel",
    "approver_name": "Sarumathi S",
    "requester_name": "John Doe",
    "request_type": "test",
```
